Etiquetas semánticas



El lenguaje HTML es un lenguaje de marcas o etiquetas, es decir que los elementos que conforman la página web se describen mediante palabras especiales que marcan el inicio y el final de los mismos, a estas palabras las llamamos etiquetas. Generalmente las etiquetas se indican por pares, es decir, etiqueta de apertura y etiqueta de cierre


HTML5 incorpora nuevas etiquetas que permiten hacer la estructura de la página web mas lógica y funcional, a estas etiquetas se les llama semánticas, pues dan una noción del tipo de contenido que englobarán y su objetivo es evitar el Excel de etiquetas div. Estas etiquetas son:


  • Header: define la cabecera de la página
  • Footer: Define pie de página
  • Nav: define un elemento de navegación en la página
  • Section: encierra el código correspondiente a una sección genérica dentro de un documento
  • Aside: delimita contenido que no es directamente principal de la página en concreto
  • Article: define artículos como noticias, entradas de un blog, etc.
