Sitio web para el aprendizaje de las etiquetas semánticas, listas y enlaces



El propósito de este sitio web es poner a disposición de los estudiantes un recurso educativo digital o un objeto virtual de aprendizaje, para dinamizar y optimizar la enseñanza sobre temas tan importantes como las etiquetas semánticas, las listas, los enlaces y demás conceptos que se deben tener en cuenta al estudiar cada uno de estos temas:


Etiquetas semánticas: son palabras especiales que marcan el inicio y el final de los elementos que conforman una página web. Entre las etiquetas semánticas podemos encontrar header, footer, nav, section, article y aside.


Listas: existen 2 tipos de etiquetas para definir una lista ("ol" y "ul"), para definir los elementos de la lista se utiliza la etiqueta "li"


Enlaces: permiten que al hacer click sobre un elemento de la página web, permite al usuario acceder a otra página o url


Para mas información sobre estos temas, puedes navegar por nuestro menú superior y revisar los diferentes contenidos
