Enlaces


Un enlace o link, puede ser un texto o una imagen en la página web, que permiten al usuario hacer click sobre este elemento para acceder a otra página o url. Para crear enlaces utilizamos la etiqueta “a” y con el parámetro href=”[enlace]” introducimos en enlace al cual se desea redirigir.
